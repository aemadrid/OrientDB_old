OrientDB is growing very fast. The issues list is every day longer with many cool features that are waiting to be introduced in: http://code.google.com/p/orient/issues/list

Below the roadmap with the main issues divided by release.

Release 1.0rc8 (Begin of January 2012):

  • High-Availability supporting Multi-Master replication
  • Open issues: http://code.google.com/p/orient/issues/list?q=label:v1.0rc8

Release official 1.0 (End of January 2012):

  • Execution of synchronous and asynchronous scripts
  • JDBC driver
  • All the known bugs for 1.0 fixed
  • Open issues: http://code.google.com/p/orient/issues/list?q=label:v1.0

Release 1.1 (March 2012):

  • Views
  • SQL sub-queries
  • Security at record level, like Unix file system
  • Pessimistic Transactions
  • Inverse relationships = Referential Integrity

Release 1.2 (May 2012):

  • Stored procedure
  • Multi master replication
  • Time machine
