Import database

Import a database to the current one opened. The input file is in JSON format using the Export format generated by ConsoleCommandExport tool. By default the file is compressed using the GZIP algorithm. The Export/Import commands allow to migrate between engine releases without loosing data.

Use the Export command to generate the JSON file to import.

Syntax

import database <input-file>

Where:

  • input-file Input file path

Example 

> import database C:\temp\petshop.export
