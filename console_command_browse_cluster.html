Browse cluster

This command display all the records of a cluster.

Syntax

browse cluster <cluster-name>

Where:

  • cluster-name The name of the cluster

Example 

> browse cluster City

---+--------+-------------------
  #| REC ID |NAME
---+--------+-------------------
  0|    -6:0|Rome
  1|    -6:1|London
  2|    -6:2|Honolulu
---+--------+-------------------

This is a command of the Orient console. To know all the commands go to Console commands.
