The SQL Rebuild Index command rebuilds an automatic index.

Syntax

REBUILD INDEX <index-name>

Where:

  • index-name name of the index. Use to rebuild all the automatic indices

Examples 

REBUILD INDEX Profile.nick

REBUILD INDEX *
